# betop_teleop 手柄遥控包说明文档

本包提供了北通鲲鹏 20（BTP-KP20D）手柄在 BFM 模式下的解析节点，包含**串口直接控制（直控模式）**与**话题中转控制（话题模式）**两个独立节点及各自的启动脚本。

---

## 1. 手柄物理按键映射与控制逻辑

手柄在运行期间执行三级安全冗余机制：

| 按键功能             | 手柄物理键      | 触发模式 | 行为控制逻辑                                                           |
| :------------------- | :-------------- | :------- | :--------------------------------------------------------------------- |
| **使能按键 (LB/RB)** | **Button 7**    | 长按使能 | 必须按住使能键，摇杆及升降控制量才允许下发；松开时速度与打角强制归零。 |
| **紧急避险急停 (B)** | **Button 1**    | 单击触发 | 触发系统立即急停，给底盘断电锁定并重置 PID 缓存。                      |
| **线速度前/后**      | **左摇杆 Y 轴** | 线性比例 | 满偏行程对应$\pm 1.2\text{ m/s}$。                                     |
| **转向左/右**        | **右摇杆 X 轴** | 线性比例 | 话题模式对应角速度，直控模式直接对应打角度数。                         |
| **千斤顶上升**       | **Y 键**        | 长按触发 | 长按周期下发上升点动信号。                                             |
| **千斤顶下降**       | **A 键**        | 长按触发 | 长按周期下发下降点动信号。                                             |

---

## 2. 独立节点一：串口直接控制节点 (`diuniu_teleop_serial`)

该节点直接抢占并连接物理串口，将手柄动作直接封装为 **14 字节二进制下行包** 写入串口以操作电机。

- **适用场景**：底盘驱动包 `diuniu_base` 未运行时，用于纯手柄硬件调试或应急遥控。
- **命令行参数**：
  - `serial_port`：串口设备路径（默认 `/dev/ttyUSB0`）。
  - `baud_rate`：串口波特率（默认 `460800`）。
  - `max_linear_speed`：最大速度限幅（默认 `1.2` m/s）。
  - `max_steer_angle`：最大前轮转向度数（默认 `95.0`°）。
- **运行命令**：
  ```bash
  ros2 launch betop_teleop diuniu_teleop_serial.launch.py port:=/dev/ttyACM0
  ```

---

## 3. 独立节点二：话题中转控制节点 (`diuniu_teleop_cmd_vel`)

该节点**不占用、不操作任何物理串口**，只读取手柄按键并向 ROS 2 话题发布速度。

- **适用场景**：标准导航与实车联调开发模式。与 `diuniu_base_node` 联合部署。
- **发布话题**：
  - **`/cmd_vel_joy`** (`geometry_msgs/msg/Twist`)
    - `linear.x` $\rightarrow$ 目标线速度 (m/s)
    - `angular.z` $\rightarrow$ 目标角速度 (rad/s)
    - `linear.z` $\rightarrow$ 升降（`1.0` 上升 / `-1.0` 下降 / `0.0` 停止）
    - `angular.x` $\rightarrow$ 急停（`1.0` 锁定 / `0.0` 正常释放）
- **运行命令**：
  ```bash
  ros2 launch betop_teleop diuniu_teleop_cmd_vel.launch.py
  ```

---

## 4. 参数微调配置

在 `launch/` 目录下的启动脚本中，支持微调摇杆的通道、死区、最大线速度及角速度上限。

- 话题模式默认配置的 `max_angular_speed` 为 `2.5 rad/s`。
- 直控模式默认配置的 `max_steer_angle` 为 `95.0 度`。
